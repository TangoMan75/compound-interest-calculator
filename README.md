Compound Interest Calculator
============================

A React-based Single Page Application (SPA) that calculates compound interest and visualizes savings growth over time through an interactive graph, using Vite, React, Tailwind, and Recharts.

🌐 Live Demo
------------

- https://tangoman75.github.io/compound-interest-calculator

🚀 Features
-----------

### ⚡ Core Functionality

1. **Compound Interest Calculation:** Calculate future balance with compound interest using the formula: Future Balance = Principal × (1 + Rate/100)^Years.
2. **Interactive Growth Graph:** Visualize savings growth over time using Recharts.
3. **Real-time Updates:** Graph updates instantly when inputs change.

### ⚡ User Experience

1. **Form Validation:** Input validation with error messages for invalid entries.
2. **Mobile Responsive Design:** Fully responsive layout for all screen sizes.
3. **PWA Ready:** Progressive Web App configuration for offline usage.

📦 Installation
---------------

1. **Install dependencies:** Run `npm install` to install all required packages.
2. **Start development server:** Run `npm run dev` to start the development server.
3. **Open in browser:** Navigate to `http://localhost:5173` to use the application.
4. **Run linter:** Run `npm run lint` to check code quality.

🛠️ Usage
--------

1. **Enter starting amount:** Input your initial investment amount (must be >= 0).
2. **Set interest rate:** Enter the annual interest rate as a percentage.
3. **Set duration:** Specify the investment period in years (must be >= 1).
4. **View results:** The calculated future balance and growth graph will display automatically.

🖇️ Dependencies / Requirements
------------------------------

**Compound Interest Calculator** requires the following dependencies:

1. React 18
2. Vite
3. Tailwind CSS 4
4. Recharts (for graphs)
5. Vitest (testing)
6. @testing-library/react

🐞 Troubleshoot
---------------

1. **Build errors:** Run `npm install` to ensure all dependencies are properly installed.
2. **Port already in use:** If port 5173 is occupied, Vite will automatically use the next available port.
3. **Graph not rendering:** Ensure all input values are valid numbers within acceptable ranges.

🧪 Testing Strategy
-------------------

1. **Run tests:** Execute `npm test` to run the test suite.
2. **Watch mode:** Use `npm test -- --watch` for continuous testing during development.
3. **Coverage:** Tests cover component rendering, calculation logic, and form validation.
